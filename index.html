Coveo UI StyleGuide

Coveo StyleGuide is a collection of UI components used in Coveo's Cloud Services administration console. All components and their documentation is available here Source code is available on GitHub

It is widely inspired by Bootstrap 3.0 and borrows some of it components.

The project is at it's early stages, some components can still have lots of breaking changes.

Compiling

Make sure you have Node and NPM installed. Run npm install, then npm run compile. The compiled styleguide will be available in the /styleguide folder.
