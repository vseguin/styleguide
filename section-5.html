5 Forms

Covers styles used for forms, such as the <input> and <select> elements.

5.1 Text Input

Below are the text-oriented form elements used on the site.

Single-Line Text Boxes, Standard text boxes.

:disabled

When disabled, the input's appearance should reflect as such.

<input type="text" class="{$modifiers}" value="Text"/>
<input type="text" class="{$modifiers}" placeholder="Placeholder"/>

5.2 Numeric Spinner

Spinner for numeric values

 
 <div class="input-group admin-numeric-spinner">
     <input type="text" value="1" class="">
     <span class="add-on">
         <div id="SpinnerUp">
             <i class="coveo-sprites-misc-spinner_arrow_up"></i>
         </div>
         <div id="SpinnerDown">
             <i class="coveo-sprites-misc-spinner_arrow_down"></i>
         </div>
     </span>
 </div>

5.3 Checkboxes

Custom checkboxes. Compatible IE9+.

:checked

Checkbox is checked

.indeterminate

Checkbox state is indeterminate

:disabled

Checkbox is not clickable/modifiable

<label class="coveo-checkbox-label">
  <input type="checkbox" class="coveo-checkbox {$modifiers}">
  <span></span>
</label>

5.4 Slide Toggle

Another style for a checkbox with stylish animations. Compatible IE9+.

:checked

Checkbox is checked

:disabled

Checkbox is not clickable/modifiable

<label class="coveo-slide-toggle-label">
  <input type="checkbox" class="coveo-slide-toggle {$modifiers}">
  <span></span>
  <span class="toggle-label">Simple text</span>
</label>

5.5 Slide Toggle Double

Slide toggle with 2 options. Compatible IE9+.


:checked

Checkbox is checked


:disabled

Checkbox is not clickable/modifiable


  <label class="coveo-slide-toggle-double first" for="choice">Choice 1</label>
  <input type="checkbox" class="coveo-slide-toggle-double {$modifiers}" name="choice"><span></span>
  <label class="coveo-slide-toggle-double second" for="choice">Choice 2</label>
  <br>
