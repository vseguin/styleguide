3 Buttons

Covers styles for buttons. They are all built from the same base class. You can use both link or button element.

3.1 Button Tags

Use the btn class on an <a> or <button> element.

Link 
<a class="btn">Link</a>
<button type="button" class="btn">Button</button>

3.2 Button Options

Use any of the available button classes to quickly create a styled button.

<button type="button" class="btn">Default</button>
<button type="button" class="btn btn-primary">Action</button>
<button type="button" class="btn btn-secondary">Info</button>
<button type="button" class="btn btn-danger">Danger</button>

3.3 Button Slim

Use the btn-slim class on an <a> or <button> element. Use any of the available button classes to quickly create a slim styled button.

Link 
<a class="btn-slim">Link</a>
<button type="button" class="btn-slim">Default</button>
<button type="button" class="btn-slim btn-primary">Action</button>
<button type="button" class="btn-slim btn-secondary">Info</button>
<button type="button" class="btn-slim btn-danger">Danger</button>

3.4 Buttons States

:hover

Highlight the button when hovered.

:disabled

Make the button change appearance to reflect it being disabled.

<button type="button" class="btn btn-primary {$modifiers}">Action</button>
